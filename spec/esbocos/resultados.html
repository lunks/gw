planeta, uma espécie de "imposto"

Os alertas de combate somente aparecerão após todas as facções concluirem seus movimentos e intenções de ataques. As intenções de ataques são
simplesmente um movimento para planeta inimigo. Note que se for 1 combate 1x1x1 aparecerá o defensor sendo atacado 2 vezes. Somente após todos
os combates forem concluídos e registrados, é que o turno termina e inicia-se o próximo, abrindo-se as opções compras etc.

Aqui o sistema lista as frotas inimigas que se encontram no mesmo planeta, ou seja, um combate. Um dos Almirantes terá acesso para registrar as perdas de todas facções. Os oponentes devem ter acesso aos dados registrados e validar ou não. Se não validado volta para preencher novamente.
As capturas só funcionam para CS, Instalações e Transp Pesados e obviamente para aqueles q não foram destruidos, elas passam para a facção capturante.
As fugas funcionam para todas as unidades. Elas simbolizam que a facção não jogou com elas. Isso gera um custo para o fujão de 40% e essas unidades voltam ao planeta origem ou vão para o planeta mais proximo.
